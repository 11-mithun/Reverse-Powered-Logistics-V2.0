## Using UCI Online Retail II

```python
import pandas as pd

# Load
df = pd.read_excel('datasets/Online Retail II.xlsx', sheet_name='Year 2010-2011')

# Filter returns (negative quantity = return/cancellation)
returns = df[df['Quantity'] < 0].copy()
returns['return_reason'] = 'Customer return'
returns['product_price'] = df['Price'].abs()
returns['days_since_purchase'] = 7  # estimate

print(f"Total returns: {len(returns)}")
print(returns.head())
```

## Batch Upload to Platform

After downloading, use the batch CSV uploader:
1. Dashboard → Returns → Batch Upload
2. Download sample template from `/api/datasets/sample`
3. Format your data to match the template
4. Upload — all returns get AI-classified automatically
